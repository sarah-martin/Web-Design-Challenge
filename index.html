Weather Summary

This project analyzed world weather data to better understand how weather changes as you approach the equator. Pulling information for 500 different cities from the OpenWeatherMap API provided sufficient data to assemble a reliable dataset and make a well-informed analysis. The specific elements looked at included temperature, humidity, cloudiness, and wind speed.

Look through the data collections found on this website to learn about the final analysis and data engineering process. Is life better around the equator?
